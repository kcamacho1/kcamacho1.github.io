Flask To-Do List App

A responsive and intuitive task management application built using Flask, Python, and Tailwind CSS, demonstrating full CRUD functionality and clean UI design.

Role: Developer | Focus: Backend Logic, Frontend Design, Database Integration

Tech Stack

Python Flask SQLite Tailwind CSS HTML5

Overview

This project is a fully functional To-Do List web app that allows users to create, read, update, and delete tasks. It was built to strengthen my skills in Python web development and demonstrate my ability to integrate backend logic with a clean, responsive frontend interface.

Key Features

  • Full CRUD functionality — add, edit, mark complete, and delete tasks.
  • Flask-based backend with Python routing and Jinja2 templating.
  • SQLite database for lightweight and persistent data storage.
  • Responsive Tailwind CSS design for desktop and mobile devices.
  • Clean UI with intuitive task interaction and minimal clutter.
