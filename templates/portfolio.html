A responsive, user-friendly portfolio site designed to showcase my projects, skills, and experience in software development, AI, and technology solutions.

Tech Stack

HTML5 CSS3 Tailwind CSS JavaScript Git & GitHub Pages

Overview

My personal portfolio is designed as a central hub to present my development work, technical skills, and professional experience. I built it from scratch to ensure complete control over the design, layout, and performance. The goal was to create a site that is both visually appealing and easy to navigate, allowing visitors to quickly explore my work and get in touch.

Key Features

  • Fully responsive design that adapts to desktop, tablet, and mobile devices.
  • Tailwind CSS for rapid, modern, and consistent styling.
  • JavaScript-enhanced navigation for smooth page transitions.
  • Reusable HTML partials for the header and footer for easier maintenance.
  • Optimized for performance and accessibility.
